TODO APP

Este proyecto es una lista de tareas que permite agregar un título y una descripción a cada tarea. También ofrece la opción de editar y eliminar las tareas existentes clasificadas mediante ID.

Responsabilidades

Como Analista de sistemas, use Javascript para crear una interfaz de usuario atractiva en el frontend En el lado del backend, utilice php para la conexion a la base de datos y la visualizacion de los tareas . Además, implementé AJAX para asegurar la asincronía entre tareas, permitiendo que las acciones se realicen sin necesidad de recargar la página. La informacion de los datos fue proveniente de una tabla MySql normalizada.

Tecnologias

Javascript | Bootstrap | Ajax | Mysql | PHP
