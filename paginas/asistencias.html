ASISTENCIAS

Mi proyecto de asistencias es un sitio web dedicado a la toma de turnos, donde usuarios y administradores pueden ver informacion detallada de los turnos, roles, empleados y usuarios

Responsabilidades

Como Analista de sistemas, use Javascript para crear una interfaz de usuario atractiva en el frontend junto al desarrollo de Datatables para la eficaz implementacion de tablas. En el lado del backend, utilice php para la conexion a la base de datos y la visualizacion de los datos de emplados con sus turnos. La informacion de los datos fue proveniente de una tabla MySql normalizada.

Tecnologias

Javascript | Bootstrap | DataTables | Mysql | PHP
